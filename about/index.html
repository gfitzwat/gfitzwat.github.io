About Us

This static website theme is lightning fast!

In my quest for speed, I wanted to build a light weight responsive static website starter theme, without using any web fonts, CSS or JavaScript frameworks. Well, I built it and it's under... 7K! The CSS is weighing in around 3K, the JavaScript is under 1K and the HTML is about 2k. (Obviously, I'm not counting the logo or images in the basic framework)

The theme is based on the Golden Ratio and I'm only using JavaScript to toggle the navigation, the HTML and CSS is valid.

I'm releasing this as open source, so feel free to download the basic theme on GitHub and make it your own.
